# 文本编辑器Vim

### 移动光标
```
# 跳到第一行
    gg
# 跳到最后一行
    G
# 跳到行首
    0 或者 ^
# 跳到行尾
    $
# 跳到第5行
    5gg
# 在当前行下面另起一行, 并变成Insert模式
    o
```
### 删除复制
```
# 删除光标所在行
    dd
# 复制光标所在行
    yy
# 粘贴粘贴板的内容到当前行的下面
    p
# 粘贴粘贴板的内容到当前行的上面
    P
```
### 插入
```
# 在当前行上面另起一行, 并变成Insert模式
    O
# 在当前行下面另起一行, 并变成Insert模式
    o
```
